
The following is a list of third-party software that has enabled the creation of FX Player.

#### Application User Interface

[JavaFX](https://openjfx.io/)

[MaterialFX](https://github.com/palexdev/MaterialFX)

[FX-BorderlessScene](https://github.com/goxr3plus/FX-BorderlessScene)

[ControlsFX](https://github.com/controlsfx/controlsfx)

[MDFX Markdown renderer for JavaFX](https://github.com/JPro-one/markdown-javafx-renderer)

#### Media playback and parsing

[LibVLC](https://www.videolan.org/vlc/libvlc.html)

[VLCJ Java framework for VLC Media Player](https://github.com/caprica/vlcj)

[FFmpeg](https://ffmpeg.org/)

[JavaCV](https://github.com/bytedeco/javacv)

[Jaffree FFmpeg command line wrapper](https://github.com/kokorin/Jaffree)

#### Subtitles

[SRTParser](https://github.com/gusthavosouza/SRTParser)

[OpenSubtitles](https://www.opensubtitles.org/)

[Java library for OpenSubtitles](https://github.com/wtekiela/opensub4j)


#### Media information

[The Movie Database - TMDb](https://www.themoviedb.org/)

[TMDb Java wrapper](https://github.com/UweTrottmann/tmdb-java)

[Discogs music database](https://www.discogs.com/)

[Discogs-client-4j](https://bitbucket.org/kristof_debruyne/discogs-client-4j/src/master/)

#### Logging

[Log4J](https://logging.apache.org/log4j/2.x/)

[SLF4J](https://www.slf4j.org/)

[Logback](https://logback.qos.ch/)
